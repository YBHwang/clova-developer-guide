TransportationRoute

TransportationRoute template

Provides route directions for public transportations. It is used to display public transportation routes on a screen. TransportationRoute has routes types as follows. For each route type, the trafficType object has different field configuration.

Section type Type description Fields of the subPath object
Subway section(1) Displays routes that use subways only. lane, name, endID, endName, endX, endY, startID, startName, startX, startY, subwayCode
Bus section(2) Displays routes that use buses only. busID, busNo, endID, endName, endX, endY, startID, startName, startX, startY, type
Walking section(3) Displays routes that use both subways and buses. distance, sectionTime

Note!

To display public transportation routes with this template, you must understand how to use the LINE map API. Use data returned from the template with the LINE map API to display directions on a map. See LINE map API documentation for more details on the NAVER map API. See Screen UI example on how route directions are displayed.

  • If this field does not exist, the section field contains interpolated points for drawing bus/subway sections.

 No

pathType StringObject An object containing the route type of the direction. The pathType.value field has the following values.

  • "2": Routes that use buses only
  • "3": Routes that use both subways and buses
 Yes

startName StringObject An object containing the name of the departure point Yes
subPath[] object array An object array containing details of sections on each route Yes

subPath[].endX StringObject An object containing X coordinates of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].endY StringObject An object containing Y coordinates of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].lane object An object containing details of lanes in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].lane.busID StringObject An object containing bus IDs. This field is used to show bus section details (when subPath[].trafficType.value is set to "2"). No
subPath[].lane.busNo StringObject An object containing bus numbers. This field is used to show bus section details (when subPath[].trafficType.value is set to "2"). No
subPath[].lane.name StringObject An object containing subway lanes. This field is used to show subway section details (when subPath[].trafficType.value is set to "1"). No
subPath[].lane.subwayCode StringObject An object containing subway codes. This field is used to show subway section details (when subPath[].trafficType.value is set to "1"). No
subPath[].lane.type StringObject An object containing bus types. This field is used to show bus section details (when subPath[].trafficType.value is set to "2"). No
subPath[].sectionTime StringObject An object containing an estimated travel time at a given section. Unit is minute. This field is used to show walking section details (when subPath[].trafficType.value is set to "3"). No
subPath[].startID StringObject An object containing IDs of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].startName StringObject An object containing names of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].startX StringObject An object containing X coordinates of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].startY StringObject An object containing Y coordinates of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (when subPath[].trafficType.value is set to "1" or "2"). No
subPath[].trafficType StringObject Section type. It is an identifier for distinguishing travel methods at a given section. Depending on the value of this field, configuration of sub objects in the subPath field can change. Available values are:
  • "1": Subway section
  • "2": Bus section
  • "3": Walking section
 No
SubwayStationCount StringObject An object containing the total number of subway stations to pass through when traveling by subway Yes
totalDistance StringObject An object containing a total travel distance. Unit is meter. Yes
totalStationCount StringObject An object containing the total number of bus stops and subway stations to pass through when traveling by bus and subway Yes
totalTime StringObject An object containing an estimated total travle time. Unit is minute. Yes
type string A content template delimiter. The value is always "TransportationRoute". Yes
