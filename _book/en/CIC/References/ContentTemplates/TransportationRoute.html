TransportationRoute

TransportationRoute template

Provides route directions for public transportations. It is used to display public transportation routes on a screen. TransportationRoute has routes types as follows. For each route type, valid fields for the trafficType object can vary.

Section type Type description Valid fields (subPath object)
Subway section(1) Displays routes that use subways only. lane, name, endID, endName, endX, endY, startID, startName, startX, startY, subwayCode
Bus section(2) Displays routes that use buses only. busID, busNo, endID, endName, endX, endY, startID, startName, startX, startY, type
Walking section(3) Displays routes that use both subways and buses. distance, sectionTime

Note!

To display public transportation routes with this template, you must understand how to use the LINE map API. Use data returned from the template with the LINE map API to display directions on a map. See LINE map API documentation for more details on the NAVER map API. See Screen UI example on how route directions are displayed.

Template field

Field name Type Field description
appLinkUrl URLObject An object containing a URL which redirects to a map app
boundary StringObject An object representing a rectangular area (MBR, Minimum Bounding Rectangle) that includes all of interpolated points, in the form of "left,top,right,bottom" string.
busStationCount StringObject The total number of stops to pass through when traveling by bus
end LocationObject An object containing coordinates of the destination point
endName StringObject An object containing the name of the destination point
linkUrl URLObject An object containing a URL which redirects to a web map
lanes[] object array An object containing interpolated points on all routes found
lanes[].create StringObject An object that decides the section type of the object.
  • If this field does not exist, the section field contains interpolated points for drawing bus/subway sections.
  • If this field does exist and create.value is set to "1", the section field contains interpolated points for drawing polylines.
lanes[].section[] LocationObject array An object array containing interpolated points for drawing bus/subway sections or polylines between sections.
pathType StringObject An object containing the route type of the direction. The pathType.value field has the following values.
  • "1": Routes only used by the subways
  • "2": Routes only used by the buses
  • "3": Routes both subways and buses use
start LocationObject An object containing coordinates of the departure point
startName StringObject An object containing the name of the departure point
subPath[] object array An object array containing details of sections on each route
subPath[].distance StringObject An object containing a travel distance of a given section. This field is used to show walking section details (subPath[].trafficType.value is "3"). When it is a subway or bus section (subPath[].trafficType.value is "1" or "2"), the value field of this object can have an empty string ("").
subPath[].endID StringObject An object containing IDs of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].endName StringObject An object containing names of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].endX StringObject An object containing X coordinates of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].endY StringObject An object containing Y coordinates of subway stations/bus stops to get off in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].lane object An object containing details of lanes in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), this field can be an empty object.
subPath[].lane.busID StringObject An object containing bus IDs. This field is used to show bus section details (subPath[].trafficType.value is "2"). When it is a subway section (subPath[].trafficType.value is "1"), the value field of this object can have an empty string ("").
subPath[].lane.busNo StringObject An object containing bus numbers. This field is used to show bus section details (subPath[].trafficType.value is "2"). When it is a subway section (subPath[].trafficType.value is "1"), the value field of this object can have an empty string ("").
subPath[].lane.name StringObject An object containing subway lanes. This field is used to show subway section details (subPath[].trafficType.value is "1"). When it is a bus section (subPath[].trafficType.value is "2"), the value field of this object can have an empty string ("").
subPath[].lane.subwayCode StringObject An object containing subway codes. This field is used to show subway section details (subPath[].trafficType.value is "1"). When it is a bus section (subPath[].trafficType.value is "2"), the value field of this object can have an empty string ("").
subPath[].lane.type StringObject An object containing bus types. This field is used to show bus section details (subPath[].trafficType.value is "2"). When it is a subway section (subPath[].trafficType.value is "1"), the value field of this object can have an empty string ("").
subPath[].sectionTime StringObject An object containing an estimated travel time at a given section. Unit is minute. This field is used to show walking section details (subPath[].trafficType.value is "3"). When it is a subway or bus section (subPath[].trafficType.value is "1" or "2"), the value field of this object can have an empty string ("").
subPath[].startID StringObject An object containing IDs of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].startName StringObject An object containing names of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].startX StringObject An object containing X coordinates of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].startY StringObject An object containing Y coordinates of subway stations/bus stops to get on in a given section. This field is used to show subway or bus section details (subPath[].trafficType.value is "1" or "2"). When it is a walking section (subPath[].trafficType.value is "3"), the value field of this object can have an empty string ("").
subPath[].trafficType StringObject Section type. It is an identifier for distinguishing travel methods at a given section. Depending on the value of this field, configuration of sub objects in the subPath field can vary. Available values are:
  • "1": Subway section
  • "2": Bus section
  • "3": Walking section
SubwayStationCount StringObject An object containing the total number of subway stations to pass through when traveling by subway
totalDistance StringObject An object containing a total travel distance. Unit is meter.
totalStationCount StringObject An object containing the total number of bus stops and subway stations to pass through when traveling by bus and subway
totalTime StringObject An object containing an estimated total travle time. Unit is minute.
type string A content template delimiter. It has an "TransportationRoute" value.
