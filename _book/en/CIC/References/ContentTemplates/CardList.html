CardList

CardList template

Displays data in a card list format on a screen. CardList has several card types. For each card type, valid fields for the card object can vary.

Card type Type description Valid fields (card object)
Type1 Displays content's image, title, and description. description, imageUrl, referenceText, referenceUrl, title
Type2 Displays content's image, title, description, and link. description, imageUrl, linkUrl, referenceText, referenceUrl, title
Type3 Displays video content. description, imageUrl, referenceText, referenceUrl, title, videoUrl
Type4 Displays news content. description, press, pressIconUrl, publishDate, title

Note!

See Screen UI example for available display formats for each card type.

Template field

Field name Type Field description
cardList[] object array An object array that displays a list of cards
cardList[].description StringObject array An object array containing the description of the content
cardList[].imageUrl URLObject An object containing the URL of the image to display. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].linkUrl URLObject An object containing the URL of the content. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].press StringObject An object containing the name of the press. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].pressIconUrl URLObject An object containing the URL of the press icon. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].publishDate DateObject An object containing the article publishing date. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].referenceText StringObject An object containing text information of the source. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].referenceURL URLObject An object containing the URL of the source. Depending on the card type, the value field of this object can have an empty string ("").
cardList[].title StringObject An object containing the title of the content
cardList[].videoUrl URLObject An object containing the URL of the video to play. Depending on the card type, the value field of this object can have an empty string ("").
subType string A card type delimiter. You can specify one of the four types.
  • Type1
  • Type2
  • Type3
  • Type4

Note!

Type1, Type2, Type3 are displayed as an empty string at the moment. You must determine the type by checking the fields of the card object.

 Yes
type string A content template delimiter. It has an "CardList" value.
