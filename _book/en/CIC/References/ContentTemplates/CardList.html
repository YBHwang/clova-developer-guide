CardList

CardList Template

Displays data in a card list format. CardList has several card types. Each card type has different field configuration for the card object.

Card type Type description card object fields
Type1 Displays content's image, title, and description. description, imageUrl, referenceText, referenceUrl, title
Type2 Displays content's image, title, description, and link. description, imageUrl, linkUrl, referenceText, referenceUrl, title
Type3 Displays video content. description, imageUrl, referenceText, referenceUrl, title, videoUrl
Type4 Displays news content. description, press, pressIconUrl, publishDate, title

Note!

See Screen UI example for available display formats for each card type.

Template field

Field name Type Field description Required
cardList[] object array An object array that displays a card list Yes
cardList[].description StringObject array An object array that contains content's description No
cardList[].imageUrl URLObject An object that contains an image URL No
cardList[].linkUrl URLObject An object that contains a content URL No
cardList[].press StringObject An object that contains a press name No
cardList[].pressIconUrl URLObject An object that contains a press icon URL No
cardList[].publishDate DateObject An object that contains an article publishing date No
cardList[].referenceText StringObject An object that contains source text information No
cardList[].referenceURL URLObject An object that contains a source URL No
cardList[].title StringObject An object that contains a content title No
cardList[].videoUrl URLObject An object that contains a video URL No
subType string Card type delimiter. You can specify one of the four types.
  • Type1
  • Type2
  • Type3
  • Type4

Note!

Type1, Type2, Type3 are displayed as an empty string at the moment. You must determine the type from field configuration of the card object.

 Yes
type string Content Template delimiter. The value is always "CardList". Yes
