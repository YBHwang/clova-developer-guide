Text

Text template

The Text template is used in providing text for the client to display on the client's screen. The name of the template fields provides style information as to how the text is composed or displayed. For example, the given text may consist of a number of paragraphs or is to be displayed as a series of tables.

Note!

See UI examples of the Text template used in display.

Template fields

Field name Type Description
bgUrl URLObject The URL of the image to display in the background. An empty string ("") indicates that background image information is unavailable.
emotionCode StringObject The code for Clova's emotion for the client to "express". If the client does not support emotional expression, ignore this field.

Note!

Contact us to find more about the emotion code specification.
highlightText StringObject or NumberObject Contains a text or number to be displayed with an emphasis. The number may contain commas as thousand separators. An empty string ("") or null indicates that there is no text or number to display with an emphasis.
imageUrl URLObject The URL of the image. An empty string ("") indicates that image is unavailable.
linkUrl URLObject The URL to a Web map, if this template contains an image of a map. An empty string ("") indicates that a URL to a Web map is unavailable.
mainText StringObject Contains the main text to display. An empty string ("") indicates that this information is unavailable.
motionCode StringObject The code for a motion for the client to perform. If the client does not support motions, ignore this field.

Note!

Contact us to find more about the motion code specification.
paragraphText StringObject Contains paragraphs to display. An empty string ("") indicates that paragraphical text is unavailable.
referenceText StringObject The name or description of the source of the content. An empty string ("") indicates that source information is unavailable.
referenceURL URLObject The URL of the content source. An empty string ("") indicates that source information is unavailable.
sentenceText StringObject Contains a sentence to display.
subText StringObject Contains supplementary text to display. An empty string ("") indicates that supplementary text is unavailable.
tableList[] object array Contains text units to be displayed in tables. All tables consist of a single column and two or three rows.
tableList[].item1 StringObject The text for the first row. An empty string ("") indicates that the first row is empty or no text is to be presented in a table.
tableList[].item2 StringObject The text for the second row. An empty string ("") indicates that the second row is empty or no text is to be presented in a table.
tableList[].item2Link URLObject or PhoneNumberObject The link or phone number for the text in the second row. An empty string ("") indicates that this information is unavailable.
tableList[].item3 StringObject The text for the third row. This field is omissible.
type string The type of this template. The value is always "Text".
