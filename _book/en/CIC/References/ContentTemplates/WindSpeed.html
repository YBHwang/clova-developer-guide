WindSpeed

WindSpeed template

The WindSpeed template is used in providing wind speed information for the client to display on the client's screen.

Note!

See a UI example on how the WindSpeed template is used in display.

Template fields

Field name Type Description
bgClipUrl URLObject The URL of the video file to play in the background.

Caution!

Due to a license issue, you are not permitted to use this URL.
linkUrl URLObject The link to open when the wind speed information is tapped. An empty string ("") indicates that this information is unavailable.
location StringObject The location this wind speed information is for. An empty string ("") indicates that this information is unavailable.
type string The type of this template. The value is always "WindSpeed".
windSpeed NumberObject The wind speed.

Template example

{
  "bgImageUrl": {
    "type": "url",
    "value": "https://ssl.pstatic.net/static/clova/service/weather/bg_cloud_night.mp4"
  },
  "location": {
    "type": "string",
    "value": "Macquarie"
  },
  "type": "WindSpeed",
  "windSpeed": {
    "type": "number",
    "value": "1m/s"
  }
}
