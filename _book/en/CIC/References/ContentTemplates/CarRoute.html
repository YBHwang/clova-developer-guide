CarRoute

CarRoute template

Provides route directions by driving. It is used to display driving routes on a screen.

Note!

To display driving routes with this template, you must understand how to use the LINE map API. Use data returned from the template with the LINE map API to display directions on a map. See LINE map API documentation for more details on the NAVER map API. See Screen UI example on how route directions are displayed.

Template field

Field name Type Field description Required
appLinkUrl URLObject An object containing a URL which directs users to a map app Yes
boundary StringObject An object representing a rectangular area (MBR, Minimum Bounding Rectangle) that includes all of interpolated points, in the form of "left,top,right,bottom" string Yes
linkUrl URLObject An object containing a URL which directs users to a web map Yes
pathList LocationObject An object array containing road segment points along the route Yes
summary object An object containing summary of driving routes Yes
summary.distance StringObject An object containing a travel distance from a departure point to a destination point. Unit is meter. Yes
summary.destination object An object containing details of the destination point Yes
summary.destination.lat StringObject An object containing the latitude of the destination point Yes
summary.destination.lon StringObject An object containing the longitude of the destination point Yes
summary.destination.name StringObject An object containing the name of the destination point Yes
summary.roadSummary[] object array An object array containing summary of roads along the route Yes
summary.roadSummary[].length StringObject An object containing the length of the road segment. Unit is meter. Yes
summary.roadSummary[].name StringObject An object containing the name of the road Yes
summary.roadSummary[].point object An object containing coordinates of the road entry point Yes
summary.roadSummary[].point.x StringObject An object containing the X coordinate of the road entry point on the NAVER map Yes
summary.roadSummary[].point.y StringObject An object containing the Y coordinate of the road entry point on the NAVER map Yes
summary.roadSummary[].roadCongestion StringObject An object containing traffic conditions on the road. summary.roadSummary.roadCongestion.value can have the following values.
  • "0": Data was not received
  • "1": Smooth
  • "2": Slow
  • "3": Stagnant
  • "4": Stopped
 Yes
summary.roadSummary[].speed StringObject An object containing an average driving speed of the road segment Yes
summary.start object An object containing details of the departure point Yes
summary.start.lat StringObject An object containing the latitude of the departure point Yes
summary.start.lon StringObject An object containing the longitude of the departure point Yes
summary.start.name StringObject An object containing the name of the departure point Yes
summary.time StringObject An object containing the estimated travel time. Unit is minute. Yes
type string A content template delimiter. The value is always "CarRoute". Yes
