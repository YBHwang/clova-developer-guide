Alarm

Alarm Template

When the user creates an alarm, CIC passes the alarm detail in Alarm template format to the client. The client should display the alarm details on the screen using the received template.

Note!

Currently, there are limits to the Alarm template as below.

  • With the voice command, the user can only request to add an alarm and check the list.
  • In order to modify or delete an alarm, the user should use the Clova app.

Template field

Field name Type Field description
repeatDay StringObject array An object array containing repeated date information if it is a weekly repeated alarm
repeatPeriod StringObject An object containing a repeated cycle. The value field of the object has the following values.
  • Empty string("") : One-time alarm
  • "daily" : Daily repeated alarm
  • "weekly" : Every week repeated alarm
scheduledTime DateTimeObject An object containing date and time of when the alarm will ring
token StringObject An object containing identifier of an added alarm.
type string A content template delimiter. It has an "Alarm" value.

Template example


// One-time alarm
{
  "type": "Alarm",
  "token": {
    "type": "string",
    "value": "072c72b9-cfc5-4127-b4fe-557a10457232"
  },
  "scheduledTime": {
    "type": "datetime",
    "value": "2017-10-09T09:00:00Z"
  },
  "repeatPeriod": {
    "type": "string",
    "value": ""
  },
  "repeatDay": []
}

// Daily repeated alarm
{
  "type": "Alarm",
  "token": {
    "type": "string",
    "value": "b5403bd0-1598-495b-a466-9385c2b1103a"
  },
  "scheduledTime": {
    "type": "datetime",
    "value": "2017-10-09T09:00:00Z"
  },
  "repeatPeriod": {
    "type": "string",
    "value": "daily"
  },
  "repeatDay": []
}

// Every week repeated alarm 
{
  "token": {
    "type": "string",
    "value": "da740e2a-01cd-4f2e-aedf-6c4285bae785"
  },
  "type": "Alarm",
  "scheduledTime": {
    "type": "datetime",
    "value": "2017-10-09T09:00:00Z"
  },
  "repeatPeriod": {
    "type": "string",
    "value": "weekly"
  },
  "repeatDay": [
    {
      "type": "string",
      "value": "monday"
    }
  ]
}
