Clova

Clova

The Clova namespace provides interfaces that return the result for user's voice request. When user requests are sent to Clova through the SpeechRecognizer.Recognize events, Clova analyzes the request, identifies what the user wants and uses the following directives to return the result for the request. The client is to process these directives and provide the Clova service user wants.

Message name Message type Message description
ExpectLogin Directive Instructs a client to prompt a user to authenticate their LINE account (i.e. login).
FinishExtension Directive Instructs a client to end the specified extension.
Hello Directive Notifies a client that a downchannel connection has been established.
Help Directive Instructs a client to provide pre-made help to the user.
RenderTemplate Directive Instructs a client to render the given template.
RenderText Directive Instructs a client to display the given text.
StartExtension Directive Instructs a client to start running the given extension.

ExpectLogin directive

Instructs a client to prompt its user to authenticate their LINE account. CIC sends this directive to a client to provide a service that requires the LINE account authentication while running in the guest mode.

Payload fields

None

Remarks

After the user gets authenticated successfully, CIC does not pick up the user's previously made request. If needed, the user has to make the request again.

Message example

{
    "directive": {
        "header": {
            "messageId": "2ca2ec70-c39d-4741-8a34-8aedd3b24760",
            "namespace": "Clova",
            "name": "RequestLogin"
        },
        "payload": {}
    }
}

  • Guest mode

FinishExtension directive

Instructs a client to end the specified extension. End the given extension when you receive this directive.

Payload fields

Field name Type Description Provided
extension string The name of the extension to finish. Always

Message example

{
  "directive": {
    "header": {
      "namespace": "Clova",
      "name": "FinishExtension",
      "messageId": "80855309-77b8-403f-bec5-b50f565d24a2",
      "dialogRequestId": "3db18cee-caac-4101-891f-b5f5c2e7fa9c"
    },
    "payload": {
      "extension": "Sample Extension"
    }
  }
}

See also

  • Clova.StartExtension

Hello directive

Notifies a client that a downchannel has been established between the client and CIC. This directive is a confirmation for the connection attempt required to use Clova.

Payload fields

None

Remarks

This directive does not have a dialog ID (dialogRequestId).

Message example

{
    "directive": {
        "header": {
            "messageId": "2ca2ec70-c39d-4741-8a34-8aedd3b24760",
            "namespace": "Clova",
            "name": "Hello"
        },
        "payload": {}
    }
}

See also

  • Connecting with CIC

Help directive

Instructs a client to give help for users. This directive is returned when a user requests help for using Clova. Provide a guide by audio or UI through the client screen when you receive this directive.

Payload fields

None

Message example

{
    "directive": {
        "header": {
            "messageId": "2ca2ec70-c39d-4741-8a34-8aedd3b24760",
            "namespace": "Clova",
            "name": "Help"
        },
        "payload": {}
    }
}

See also

None

RenderTemplate directive

Instructs a client to display data, the result of recognizing user's voice request, filled in the given content template.

Payload fields

The format of the payload field differs depending on the type of content template used. Available content templates are as follows.

  • Templates for different UI layouts

    • CardList
    • ImageList
    • ImageText
    • Popup
    • Text

  • PIMS template

    • ActionTimer
    • ActionTimerList
    • Alarm
    • AlarmList
    • Memo
    • MemoList
    • Reminder
    • ReminderList
    • Schedule
    • ScheduleList
    • Timer
    • TimerList

  • Templates for weather forecast

    • Atmosphere
    • Humidity
    • TodayWeather
    • TomorrowWeather
    • WeeklyWeather
    • WindSpeed

Message example

{
  "directive": {
    "header": {
      "namespace": "Clova",
      "name": "RenderTemplate",
      "messageId": "c7b9882e-d3bf-45b6-b6f4-2fc47d04e60e",
      "dialogRequestId": "ca10e609-1d24-4418-bc2a-21b70c5ea1a7"
    },
    "payload": {
        {{TemplateFormat}}
    }
  }
}

See also

  • Clova.RenderText
  • Content template

RenderText directive

Instructs a client to display the given text message.

Payload fields

Field name Type Description Required
text string The text message to display to a user. Required

Message example

{
  "directive": {
    "header": {
      "namespace": "Clova",
      "name": "RenderText",
      "messageId": "6f9d8b54-21dc-4811-8e14-b9b60717cbd4",
      "dialogRequestId": "5690395e-8c0d-4123-9d3f-937eaa9285dd"
    },
    "payload": {
      "text": "I will play an exciting song for you."
    }
  }
}

See also

  • Clova.RenderTemplate

StartExtension directive

Instructs a client to start the specified extension. Start the given extension when you receive this directive.

      "namespace": "Clova",

      "dialogRequestId": "8b509a36-9081-4783-b1cd-58d406205956"
    },
    "payload": {
      "extension": "Sample Extension"
    }
  }
}
